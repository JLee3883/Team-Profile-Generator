# Team-Profile-Generator
[![License: MIT](https://img.shields.io/badge/License-MIT-yellow.svg)](https://opensource.org/licenses/MIT)

## Description 
This application is a Node.js command-line team profile generator that takes in the information about employees on a software engineering team which then generates an HTML webpage that displays summaries for each person added. 

[Google Drive Link](https://drive.google.com/file/d/1eXonHW-neeyUzwFRk9zxDecZlUUnMHGE/view)

## Installation Instructions
The user must install nmp Inquirer and Jest to run and test

## Usage Information
You can quickly and easily assemble cards for your team members by running 'node index.js' in your terminal.
